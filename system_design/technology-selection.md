# 云服务技术选型
> 旨在以“0成本”或极低的成本（通常低于10元/月）搭建并运行一个功能完整的RAG问答系统，以满足竞赛的演示和开发需求。

## 核心思路

我们的目标是避免使用需要长期开启且费用较高的虚拟机（ECS），最大限度地利用阿里云各项产品的**免费额度**和**Serverless能力**。通过将项目架构拆解，并与特定的云产品免费套餐进行映射，实现近乎零成本的云上部署。

## 准备工作：获取学生权益

一切“0成本”部署的基础是获得阿里云的学生身份认证。

1.  **注册与实名**：注册阿里云账号，并完成个人实名认证。
2.  **学生认证**：访问“[高校学生“飞天加速计划”](https://developer.aliyun.com/plan/student)”页面，按照指引完成学生身份认证。认证通过后，您将获得多种云产品的免费额度或学生专属优惠价。

## 云服务技术选型方案

下表将本项目的技术组件与推荐的、有免费额度的阿里云产品进行对应：

| 组件类别 | 原定技术栈 | 云服务方案 | 免费额度说明 | 备注 |
| :--- | :--- | :--- | :--- | :--- |
| **前端部署** | 未定义 | **阿里云 OSS** | OSS提供免费存储和流出流量包 | 将Vue应用打包后直接上传至OSS Bucket，开启静态网站访问即可。 |
| **后端服务** | Python Flask + ECS | **函数计算 FC** | 每月百万次调用+40万vCPU-s免费 | 将Flask应用改造为Serverless架构，每个API对应一个函数，按需执行，无请求不花钱。 |
| **文档解析处理** | 未定义 | **文档智能 (Document Mind)** | 通常提供新用户免费试用或按量付费的免费额度 | 用于替代传统解析库，实现对PDF、图片等复杂文档的精准解析和结构化输出。 |
| **大模型推理** | Qwen + PAI-EAS | **阿里云百炼** | 提供新用户免费额度（如百万级Tokens） | 直接调用成熟的API服务，免去部署麻烦，性能更优。本项目选用 `qwen-plus`。 |
| **向量及精排模型** | 自行部署 | **PAI-EAS 免费版** | 提供1核2G CPU的免费服务实例 | 用于部署 `bge-m3` 和 `bge-reranker-v2-m3` 等轻量级模型。 |
| **向量存储** | Weaviate | **OpenSearch（向量检索版）** | 提供新用户免费试用或一定额度的免费版 | 全托管的向量检索服务，无需自行部署和维护。 |
| **知识库存储** | 本地磁盘 | **阿里云 OSS** | 提供每月50GB标准存储等免费额度 | 用于存放原始知识文档（PDF、Markdown等）。 |
| **用户数据库** | MySQL | **RDS for MySQL** | 提供新用户免费试用实例 | 如果用户量不大，也可简化为文件存储在OSS或使用表格存储。 |
| **AI工作流编排** | langgraph(langchain) | 不变 | - | 在函数计算FC中运行Python代码即可。 |
